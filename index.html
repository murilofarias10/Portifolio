    SQL

    Este banco de dados possui 7 tabelas com quase 4 mil dados e foi construido no Bootcamp Santander Fullstack Developer, então uma maneira rápida de realizar uma consulta é utilizando o comando JOIN, neste exemplo utilizei o código: select banco.numero, banco.nome, agencia.numero, agencia.nome from banco JOIN agencia ON agencia.banco_numero = banco.numero; Para trazer 4 colunas com seus respectivos valores quando a coluna banco_numero da tabela agencia for igual a coluna numero da tabela banco
